# MIT-CAVE monorepo
CAVE Open Source Application Development Repo

## Installation

Requirements:
- `node@10.16.0` or higher
- `npm@6.9.0` or higher

From a freshly-cloned repo in the project root:
```bash
npm ci
npm run bootstrap
npm run prep-link
cd ~/path/to/projects
create-cave-app new myprojectname
```
